char* inToPostfix ( char *  infix)

A function to convert infix to postfix form.

this function parse through the infix form string till the end It pushes the element if it is alphabet other than V If it is an operator it pops out the operators in the stack with higher precedence and then pushes this element to stack recursively. And finally flushes the stack

Parameters
infixthe formula in infix form
Returns
postfix which is the formula in postfix form
Note
the letter V is used for OR operation. Do not use it as a propositional atom
